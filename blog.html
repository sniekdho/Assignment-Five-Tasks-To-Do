Question-1: What are the different ways to select an element in the DOM?

There are multiple ways to select elements in the DOM.

1. getElementById() -- Selects a single element by its unique ID.
2. getElementsByClassName() -- Selects all elements with a specified class name.
3. getElementsByTagName() -- Selects all elements of a specified tag name.
4. querySelector() -- Selects the first element that matches a specified CSS selector.
5. querySelectorAll() -- Selects all elements that match a specified CSS selector.

Question-2: What is the difference between innerHTML, innerText, and textContent?

innerHTML -- Recognizes HTML tags and renders the content according to the tags.
innerText -- Reads the content as it appears on the screen, ignores the hidden content.
textContent -- Reads content as it appears in the markup. It reads the hidden content also.

Question-3: What is event delegation in the DOM?

The system of not adding click event listener for each items and only add the click event listener to all items parent is known as event delegation.

Question-4: What is event bubbling in the DOM?

In a nested elements, the click event which is triggered on the innermost element but the event is automatically triggered it's parents, grandparents and so on is known as event bubbling.

Question-5: How do you create, add, and remove elements using JavaScript?

For "create" an element using javaScript -- document.createElement('tagName')
For "add" an element using javaScript -- first create a new element which will be treated as child element, then find the parent element where to add the new element. Finally use "appendChild()" method to add the child element with parent.
For "remove" an element using javaScript -- first select the parent element, then find that element which will be removed. Finally use "removeChild()" method to remove the element from parent.
